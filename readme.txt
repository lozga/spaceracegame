Thanks for your interest. With this project I'm trying to create an interesting turn-based strategy about space race between USA and USSR from 1957 to about 1975.

Installation instructions:
Windows users:
1. Run spaceracegame.exe
2. If the above fails, manually install Java JVM:
http://java.com/en/download/index.jsp
3. Run spaceracegame.jar

Linux users:
1. Install Java JVM (if needed)
2. Run spaceracegame.jar
